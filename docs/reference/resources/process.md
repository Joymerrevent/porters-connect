# Process — フィールド／項目リファレンス

- endpoint: `/v1/process`
- scope: `process_r / process_w`
- alias 接頭辞: `Process.`（Candidate は `Person.`、Phase は短縮名）
- 出典: <https://hrbcapi.porters.jp/hc/ja/articles/115008016727-Process-Field-List>（updated_at 2022-07-15、取得 2026-06-12）

> 記事の主要テーブルを機械抽出したもの（要約・整形済み）。正確な最新は出典を参照。

## カスタム項目

- `Process.U_[Name]`（ユーザー作成）/ `Process.A_[Name]`（アプリ作成）はテナント毎に異なる。
  上表に無い項目は **Field Read API**（`/v1/field`）で動的に取得する。
