# Candidate — フィールド／項目リファレンス

- endpoint: `/v1/candidate`
- Read scope: `candidate_r, user_r, option_r`（＋ 参照する上位リソースの `_r`）
- Write scope: `candidate_w`
- alias 接頭辞: `Person.`
- 出典: フィールド <https://hrbcapi.porters.jp/hc/ja/articles/115008016747-Candidate-Field-List>（updated_at 2019-02-01）／ scope は Read・Write 記事より。取得 2026-06-12

> 記事の主要テーブルを機械抽出したもの（要約・整形済み）。正確な最新は出典を参照。

## カスタム項目

- `Person.U_[Name]`（ユーザー作成）/ `Person.A_[Name]`（アプリ作成）はテナント毎に異なる。
  上表に無い項目は **Field Read API**（`/v1/field`）で動的に取得する。
