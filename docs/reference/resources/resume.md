# Resume — フィールド／項目リファレンス

- endpoint: `/v1/resume`
- scope: `resume_r / resume_w`
- alias 接頭辞: `Resume.`（Candidate は `Person.`、Phase / Attachment は接頭辞なしの短縮名）
- 出典: <https://hrbcapi.porters.jp/hc/ja/articles/115008171848-Resume-Field-List>（updated_at 2019-02-01、取得 2026-06-12）

> 記事の主要テーブルを機械抽出したもの（要約・整形済み）。正確な最新は出典を参照。

## カスタム項目

- `Resume.U_[Name]`（ユーザー作成）/ `Resume.A_[Name]`（アプリ作成）はテナント毎に異なる。
  上表に無い項目は **Field Read API**（`/v1/field`）で動的に取得する。
