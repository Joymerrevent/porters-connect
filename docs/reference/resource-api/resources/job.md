# Job — フィールド／項目リファレンス

- endpoint: `/v1/job`
- Read scope: `job_r, recruiter_r, client_r, user_r, option_r`（＋ 参照する上位リソースの `_r`）
- Write scope: `job_w`
- alias 接頭辞: `Job.`
- 出典: [フィールド定義 記事][src]（updated_at 2022-07-15）／ scope は Read・Write 記事より。取得 2026-06-12

> 記事の主要テーブルを機械抽出したもの（要約・整形済み）。正確な最新は出典を参照。

## カスタム項目

- `Job.U_[Name]`（ユーザー作成）/ `Job.A_[Name]`（アプリ作成）はテナント毎に異なる。
  上表に無い項目は **Field Read API**（`/v1/field`）で動的に取得する。
