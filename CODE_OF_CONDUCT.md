# Contributor Covenant 行動規範

## 私たちの誓い

私たちメンバー、コントリビューター、およびリーダーは、年齢、体格、目に見える・見えない障がい、民族性、性的特徴、性自認・性表現、経験の度合い、教育、社会経済的地位、国籍、容姿、人種、宗教、性的指向にかかわらず、誰もがハラスメントのないコミュニティ参加体験を得られるようにすることを誓います。

私たちは、オープンで歓迎的、多様で、包摂的かつ健全なコミュニティに貢献する形で行動し、交流することを誓います。

## 私たちの基準

コミュニティにとって良い環境に寄与する行動の例：

- 他者への共感と優しさを示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを与え、また快く受け取る
- 自らの過ちの責任を受け入れ、影響を受けた人々に謝罪し、その経験から学ぶ
- 個人だけでなくコミュニティ全体にとって最善のことに目を向ける

容認されない行動の例：

- 性的な言葉・表現の使用、あらゆる種類の性的な誘いや関心の押し付け
- 荒らし、侮辱・中傷的なコメント、個人攻撃や政治的攻撃
- 公的・私的を問わないハラスメント
- 明示的な許可なく他者の物理的・電子的な私的情報（住所など）を公開する
- 職業上の場にふさわしくないと合理的に判断されるその他の行為

## 執行の責任

コミュニティのリーダーは、容認される行動の基準を明確化・徹底する責任を負い、不適切・脅迫的・攻撃的・有害と判断した行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

この行動規範は、コミュニティのすべての空間に適用され、また個人が公の場でコミュニティを公式に代表する場合にも適用されます。

## 執行

虐待・ハラスメント等の容認されない行動は、執行担当へ <shiromoto_jun@joymerrevent.com> まで報告してください。すべての申し立ては迅速かつ公正に確認・調査されます。
コミュニティのリーダーは、報告者のプライバシーと安全を尊重する義務を負います。
