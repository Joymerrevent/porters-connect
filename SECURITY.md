# セキュリティポリシー

`@joymerrevent/porters-connect`（**非公式** PORTERS Connect API ラッパー）の脆弱性報告窓口です。

> このライブラリは [Joymerrevent][joymerrevent] 製の**非公式**ソフトウェアで、ポーターズ株式会社とは無関係です。
> **PORTERS 製品・Connect API 本体の不具合や脆弱性は、ここではなく PORTERS 公式へ**報告してください。
> 本ポリシーの対象は**このライブラリのコード**のみです。

## 対応バージョン

最新の `0.1.x` のみをサポート対象とします（公開済みの版は上書きできないため、修正は常に新バージョンで提供します）。

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |
| < 0.1.0 | ❌        |

## 報告方法

**公開 Issue では報告しないでください。** 次のいずれかでお願いします。

1. **GitHub の private vulnerability reporting**（推奨）：リポジトリの **Security → Report a vulnerability** から非公開で報告。
2. メール：<shiromoto_jun@joymerrevent.com>

報告には、影響範囲・再現手順・想定される攻撃シナリオ・対象バージョンを含めてください。

## 対応の流れ（ベストエフォート）

- 個人メンテナンスの小規模プロジェクトのため、可能な範囲での対応となります。
- 受領後に内容を確認し、妥当な脆弱性であれば修正版を新バージョンとして公開し、GitHub Security Advisory で開示します。
- クレジット掲載の希望があればお知らせください。

[joymerrevent]: https://github.com/Joymerrevent
